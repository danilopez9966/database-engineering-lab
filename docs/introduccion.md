# Introduction

This repository is a personal database engineering lab focused on practicing SQL and database administration concepts with MySQL and PostgreSQL.

## Current section

The current MySQL section includes:

- Basic DDL (database and table creation)
- Basic DML (data insertion)
- Simple SQL queries
- Relationships between tables using foreign keys

## Goals

The main goals of this repository are:

- Improve SQL proficiency
- Practice database design
- Learn optimization techniques
- Build a solid portfolio focused on database engineering

## Next topics

Upcoming sections will include:

- JOINs and subqueries
- Views and CTEs
- Stored procedures and functions
- Triggers
- Transactions
- Indexes and optimization
- Users, roles and security
