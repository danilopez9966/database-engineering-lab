# Database Engineering Lab

A hands-on repository focused on **SQL, database design, optimization, and administration** using **MySQL and PostgreSQL**.

This project serves as a personal lab where I practice and improve my skills in database engineering, backend data handling, and performance optimization.

---

## About This Repository

This repository contains structured exercises and examples covering:

- SQL fundamentals and advanced queries
- Database design and normalization
- Stored procedures and functions
- Triggers and automation
- Transactions and data integrity
- Indexing and query optimization
- User management and security
- Backup and recovery strategies
